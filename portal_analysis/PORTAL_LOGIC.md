## 1. Типы полей

| Тип | Описание | Пример |
|---|---|---|
| `text` | Свободный текст | Название площадки |
| `number` | Число | Площадь, га |
| `dropdown` | Выпадающий список с фиксированными значениями | Преф. режим |
| `dropdown_conditional` | Выпадающий список, от которого зависят дочерние поля | Наличие ж/д |
| `child` | Поле, появляющееся только при определённом значении родителя | Расстояние до ж/д |

## 3. Что портал считает «заполненным»

> ⚠️ Уточнить по каждому типу поля.

| Тип поля | Считается заполненным | Не считается заполненным |
|---|---|---|
| `text` | Любой непустой текст | Пусто, пробелы |
| `number` | Число > 0 (?) | 0, пусто |
| `dropdown` | Любое значение из списка, включая «Нет» | Пусто |
| `dropdown_conditional` | Любое значение | Пусто |
| `child` | Зависит от типа | — |
