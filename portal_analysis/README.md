# portal_analysis/

Папка для всего, что связано с анализом заполняемости площадок на портале invest.gov.ru.

## Структура

```
portal_analysis/
├── README.md              ← этот файл
├── PORTAL_LOGIC.md        ← методика: поля, типы, условные зависимости
├── portal_checker.py      ← основной модуль проверки заполняемости
└── tests/
    └── test_portal_checker.py  ← тесты логики
```

## Как работает

1. `PORTAL_LOGIC.md` — источник истины о логике портала (заполняется вручную)
2. `portal_checker.py` — реализует эту логику в коде
3. Модуль импортируется в основное приложение через `update_routes.py`

## Версионирование

Каждое изменение логики фиксируется в `PORTAL_LOGIC.md` в разделе `## Changelog`.
